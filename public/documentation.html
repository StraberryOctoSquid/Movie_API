This is an api with information about some of my favorite movies!

Please refer to the chart below for questions about what this API can do and how to interact with it.

Business Logic URL HTTP METHOD Request Body Data Format Response Body Data Format
Return list of all movies /movies Get None A JSON object holding data about all movies in api
Return information about a director /directors[name] Get None A Json object holding data about a single director
Return information about a genre /genre/[name] Get None A JSON object holding data about a single genre
New user registration /users POST A JSON object holding the new user's name and an array containing their favorite movies, structured like: { "name": "Mary", "favoriteMovies":[] } A Status of 201 will indicate that the user was succesfully added. A status message 400 with a text of 'users need names' will indicate a failed new user registration.
Update user name /users/[id] PUT "A JSON object holding the user's new user name and an array containing their favorite movies, structured like: { ""name"": ""Kim Updated Name"", ""favoriteMovies"":[] }" "A status message of 200 will indicate a succesfully updated user name. A status messgae of 400 and a text message 'no such user' will indicate a failed attempt."
Add a movie title to a user's list of favorites /users/[id]/[movieTitle POST None "A status message of 200 accompanied by a text message of "" ${movieTitle} has been added to user ${id}'s"" will confirm that a movie title has been succesfully added to the user's profile. A status message of 400 accompanied by the text 'no such user' will indicate a failed attempt."
Remove a movie title from a user's list of favorites /users/[id]/[movieTitle] DELETE None "A status message of 200 accompanied by a text message of "" ${movieTitle} has been deleted from user ${id}'s"" will confirm that a movie title has been succesfully deleted from the user's profile. A status message of 400 accompanied by the text 'no such user' will indicate a failed attempt."
Delete a user's profile /users/[id] DELETE None "A status message of 200 and a text message 'User ${id} has been deleted' will indicate success. A status message of 400 and a text message 'no such user' will indicate a failed attempt."
